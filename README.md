# Diff Lens — Visual Text Comparator

A visual text diff tool built by [BuiltByCrew](https://www.builtbycrew.online).

**Live:** https://difflens.builtbycrew.online

Paste two versions of any text side-by-side and instantly see a color-coded visual diff highlighting every addition, deletion, and change. Perfect for comparing code snippets, draft revisions, config files, or any text you want to audit at a glance.
